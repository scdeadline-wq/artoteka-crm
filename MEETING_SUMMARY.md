## 2. CRM-МОДУЛЬ (клиенты и продажи)

### Карточка клиента
- ФИО / контакты
- Предпочтения по художникам (напр. «любит Смирнова и Русецкого»)
- История покупок
- Описание / психологический портрет
- Данные из соцсетей (опционально)
- Возраст и доп. данные (для AI-рекомендаций)
- Тип: конечный покупатель / дилер / реферал

### Рефералы
- Через кого продали (дилеры, посредники, дизайнеры)
- Реферальное вознаграждение
- Аналитика эффективности по рефералам

### Продажи
- Привязка: работа → покупатель → реферал
- Кому продали, когда, за сколько
- Маржа (цена продажи − цена закупки)

## 3. АВТОМАТИЗАЦИЯ И УВЕДОМЛЕНИЯ

### Автоматическая рассылка

- «Жамка и кнопку» — в один клик отправить предложение

### Telegram-интеграция
- **Закрытый Telegram-канал** — для дилеров (приватный, не открытый)
- **Telegram-бот** — отвечает на запросы из базы с ценами, AI-подбор
- Цель: снизить нагрузку от входящих звонков «а что у вас есть из Пивоварова?»

### AI-функции
- Автозаполнение техники по фото
- AI-бот для общения с клиентами и подбора работ
- Рекомендательный сервис на основе истории покупок
- В перспективе: MCP-доступ для AI-агентов

## 5. WORKFLOW: ВХОДЯЩИЕ РАБОТЫ

1. Человек отправляет фото + описание (через бота или сайт)
2. Работа появляется в статусе **«На рассмотрении»**
3. Алексей проверяет → нажимает **ОК**
4. Работа попадает в общую базу (только когда физически на складе)
5. Ключевое: **вещь не в базе, пока не на складе** (иначе кидают)

### Приватность B2B
- Нельзя показывать одну вещь всем дилерам — «все узнают, никто не купит»
- Всем нужен эксклюзив
- Персональные предложения, не общий каталог

## 6. БИЗНЕС-МОДЕЛЬ (на будущее)

### Маркетплейс «Артотека»
- Агрегатор антикварных / винтажных магазинов
- Парсинг баз партнёров → единый каталог
- Комиссия с двух сторон (продавец + покупатель)
- Рынок подарков с историей (советское, винтаж, новоделы)
- Бесплатные годовые подписки для первых партнёров

### Масштабирование
- Цель: «самая большая база предметов искусства в России за 2-3 месяца»
- Возможность обмена базами между галереями (выборочно, 50 работ)
- В перспективе: интерес для госкаталога музеев

### Инфраструктура
- Серверы: один российский + один зарубежный (резервный)
- ~$3000 на серверное решение
- Резервное копирование обязательно
