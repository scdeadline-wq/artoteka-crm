## 1. Общая идея

CRM-система для художественной галереи (рабочее название **ArtSpace**).
Не просто учётная система — интеллектуальная платформа, где AI помогает:
- принимать произведения (фото + голос → карточка)
- верифицировать подлинность (анализ + веб-поиск)
- готовить контент для сайта (улучшение фото, интерьерные мокапы)
- управлять художниками, коллекционерами, выставками и продажами

---

## 2. Модули CRM

### 2.1 Обзор (дашборд)
- Метрики: выручка за месяц, количество произведений, художники, активные сделки
- Топ-художники с бейджами статуса
- Ближайшие выставки
- Воронка продаж (канбан по стадиям: Запрос → Переговоры → Согласование → Сделка)

### 2.2 Художники
Поля: имя, email, направление/техника, количество работ, статус (Активен / На выставке / Эксклюзив / Архив)

### 2.3 Произведения (коллекция)
Поля: фото, название, художник, год, техника, размер (Ш×В в см), цена, валюта, статус (Свободна / Резерв / На выставке / Продана)

### 2.4 Коллекционеры
Поля: имя, email, интересы, история покупок, общая сумма

### 2.5 Продажи
Воронка + история закрытых сделок + метрики (конверсия, средний чек)

### 2.6 Выставки
Поля: название, даты, зал, куратор, художники, количество произведений, статус, посещаемость

## 3. Ключевая фича — Интеллектуальная приёмка произведений

> Это главная инновация системы. Реализована как 4-шаговый пайплайн.

### Шаг 1 — Загрузка
- Drag & drop фото (обязательно) — JPG/PNG/WEBP
- **Голосовое описание** через Web Speech API — транскрибируется в реальном времени
- Поля: размеры в см (Ш × В), техника, год, художник, цена, валюта

### Шаг 2 — AI Верификация
- Отправляем в Claude API: изображение + голосовой текст + поля
- Включён инструмент `web_search` — Claude ищет художника, аукционные аналоги, рыночную стоимость
- Возвращает JSON:
  ```json
  {
    "title": "...",
    "artist": "...",
    "nationality": "...",
    "style": "...",
    "technique": "...",
    "period": "...",
    "estimated_value_low": "...",
    "estimated_value_high": "...",
    "confidence": 0.85,
    "web_findings": [...],
    "verification_status": "verified|likely|unconfirmed",
    "conclusion": "..."
  }
  ```
- Показываем: карточку идентификации, веб-находки, полосу достоверности, экспертное заключение

### Шаг 3 — Улучшение фото + интерьеры
- **Улучшение:** оригинал vs обработанная версия (контраст, насыщенность, резкость, баланс белого)
- **Интерьерные мокапы** — 3 сцены на Canvas 2D с точными пропорциями:

  **Пропорциональность — ключевое требование:**
  - Пользователь вводит реальные размеры в см (например, 60×80 см)
  - Масштаб рассчитывается относительно реального помещения (секция стены 350×270 см)
  - `px_per_cm = canvas_width / 350` → artwork_px = artwork_cm × px_per_cm
  - Произведение располагается на уровне глаз (155 см от пола)
  - 60×80 см выглядит в интерьере иначе, чем 150×200 см — это визуально корректно

  **Три интерьера:**
  1. Современная гостиная (тёплые тона, диван, паркет, рама из дерева)
  2. Белый куб / галерея (белые стены, бетонный пол, споты на потолке, галерейный лейбл)
  3. Классический кабинет (тёмно-зелёные стены, лепнина, паркет ёлочкой, позолоченная рама, бра)

### Шаг 4 — Публикация
- Превью карточки сайта (фото, название, художник, метаданные, цена)
- Кнопки: Опубликовать на сайт / Добавить в CRM / PDF-карточка

## 4. Система AI-верификации подлинности

> Честная картина: что AI может, что нет, какие данные нужны.

### Слои данных и их вес для достоверности

| Источник данных | Вклад в достоверность | Кто делает |
|---|---|---|
| Фото высокого разрешения (6+ углов) | +35% | Галерея |
| УФ-свет + рейк-свет (под углом 5–15°) | +12% | Галерея |
| Задокументированный провенанс | +18% | Владелец |
| Запись в каталоге-резоне | +15% | Поиск AI |
| Макро-фото подписи + сравнение | +8% | Галерея |
| Лабораторный анализ материалов | +20% | Лаборатория |

**Минимальные требования к фото:**
- Разрешение: минимум 300 DPI, лучше 600–1200 DPI
- Размер: от 4000×6000 px
- Обязательные ракурсы: фронт, оборот, подпись (макро), детали фактуры, рейк-свет

### Что AI делает хорошо
- Первичный визуальный анализ стиля и техники
- Поиск аналогов в аукционных базах (Christie's, Sotheby's, artnet, Artsy)
- Выявление красных флагов и аномалий
- Структурирование досье для эксперта (снижает стоимость экспертизы в 3–5 раз)

### Что AI НЕ заменяет
- Физический анализ состава материалов (XRF, масс-спектрометрия — лаборатория)
- Юридически значимое заключение (нужна подпись эксперта)
- Тактильную оценку (живость краски, качество грунта)
- Работы малоизвестных/региональных художников (мало обучающих данных)

### Порог принятия решений для галереи
| Стоимость | Действие |
|---|---|
| До €5 000 | AI-анализ + провенанс — достаточно для внутреннего учёта |
| €5 000–50 000 | AI + консультация искусствоведа по фотоматериалам |
| От €50 000 | Обязательно лабораторный анализ + официальная экспертиза |

### TODO — PDF «Досье для эксперта»
Автоматическая генерация документа, который содержит:
- все фото произведения
- результаты AI-анализа
- найденные веб-источники
- поля для заключения эксперта
→ Эксперт получает готовое досье и работает быстрее
